🔔 Entendendo o addEventListener no JavaScript

O método addEventListener() permite registrar múltiplos manipuladores de eventos em um único elemento. Ele é muito útil para tornar as páginas interativas sem sobrescrever outras funções de evento.

📌 Exemplo de uso:


const botao = document.getElementById("meuBotao");
botao.addEventListener("click", function() {
  alert("Você clicou no botão!");
});

🎯 Teste interativo:

Este texto será alterado ao clicar no botão!
