Entendendo o método addEventListener em JavaScript

O addEventListener é um método que permite que você execute uma função sempre que um evento ocorre em um elemento HTML, como um clique, passar o mouse, entre outros.

Ele é muito útil para tornar suas páginas interativas, pois você pode associar várias ações a diferentes eventos.

Aqui está um exemplo básico de uso do addEventListener para mudar a cor de um texto ao clicar no botão.

Exemplo de código JavaScript com addEventListener


const botao = document.getElementById('mudarCorBtn');
const texto = document.getElementById('texto');

botao.addEventListener('click', () => {
    texto.style.color = 'blue';
});

Teste a interação

Este texto vai mudar de cor ao clicar no botão abaixo.
